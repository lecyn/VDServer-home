# 实用工具分享

大家好，为了方便大家日常使用，我搭建了几个实用工具。以下是每个工具的简介和使用方法。

---

## 1. FileCodeBox —— 临时文件快递

**用途**：匿名分享文件，对方通过取件码即可下载，简单安全，适合临时传递文件。

**访问地址**：  
http://106.13.165.16:12345

**使用方法**：
- **上传文件**：打开页面后，点击上传按钮选择文件，系统会自动生成一个6位取件码。
- **下载文件**：将取件码告诉对方，对方在页面输入取件码即可下载文件。
- **特点**：文件默认有效期最长1天，无需登录，完全匿名。

---

## 2. RustDesk —— 远程桌面工具

**用途**：开源的远程桌面软件，可以随时随地远程控制电脑，无需公网IP，自建中继服务器速度更快。

**使用步骤**：
1. 从 [RustDesk 官网](https://rustdesk.com/) 下载对应系统的客户端。
2. 安装并打开软件，点击左侧的 **ID/中继服务器** 设置（⚙️ 图标）。
3. 在 **ID服务器** 和 **中继服务器** 中填入：106.13.165.16（端口保持默认）
4. **Key**：在 **Key** 字段中填入群内询问获取的密钥（**API服务器** 留空即可）。
5. 保存后，即可通过软件界面中的 **ID** 和 **密码** 进行远程连接。

## 3. Screego —— 屏幕共享

**用途**：轻量级的屏幕共享服务，无需注册，打开浏览器即可分享屏幕，适合临时演示或协作。

**访问地址**：  
http://106.13.165.16:5050

**使用方法**：
- **创建房间**：打开上述地址，点击 **Create room**，输入房间名后进入。
- **分享屏幕**：进入房间后，点击 **Share your screen** 选择要分享的窗口或整个屏幕。
- **加入房间**：其他人通过同一个链接（或输入房间名）进入，即可实时观看你的屏幕。
- **注意**：房间默认不设密码，建议用完即关或只分享给信任的人。
